Haunted Mansion Escape Room Controller (Embedded Systems)

Overview: Designed and created an escape-room unlock mechanism for use on small single-board computers. Focus was set for simple error prevention and reliable feedback.

Technologies used:

  • Languages: MicroPython
  • Frameworks Machine (GPIO/PWM), Time
  • Tools: Rasberry Pi Pico, Thonny IDE, components and breadboard (buzzer, servo, push button)
decisions and interactions
  • Push button was used as main input, buzzer was used to alert users of button press and servo lock/unlock door.
  • Error prevention using short delays to stop accidentally inputing several times
  • state toggling so the door remains open until intentionally closed.
