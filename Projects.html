BackChat Social Media Project (Java)

Overview: An early iteration of a user to user social media app, including, messaging, mutual friends, friend of friend's viewing, recommendations based on users and filters for finding friends. java serialisation was used for Data persist.

Technologies used:

  • Languages: Java
  • Frameworks: Standard Java Library(Serialization, I/0, collections)
  • Tools: VScode, Github, JDK 17+
Highlights
  • Code model: User, BackChatMain, BackChat, Message
  • Contains, FOAF viewing with mutuals signified, Mutual Friendships, filering by home/work - expandable into any filters of users info.
  • Menu-driven CLI with input validation and error handling
  • Persistence to its network file and global rebuild on load

Haunted Mansion Escape Room Controller (Embedded Systems)

Overview: Designed and created an escape-room unlock mechanism for use on small single-board computers. Focus was set for simple error prevention and reliable feedback.

Technologies used:

  • Languages: MicroPython
  • Frameworks Machine (GPIO/PWM), Time
  • Tools: Rasberry Pi Pico, Thonny IDE, components and breadboard (buzzer, servo, push button)
decisions and interactions
  • Push button was used as main input, buzzer was used to alert users of button press and servo lock/unlock door.
  • Error prevention using short delays to stop accidentally inputing several times
  • state toggling so the door remains open until intentionally closed.
